# 基于java的教学过程管理APP的设计与实现

> android app 的设计与实现------毕业设计题目

## 1. 任务书

```
毕业论文(设计)任务书

基于Java的教学过程管理APP设计与实现


2．毕业论文(设计)起止时间：2019 年1月4日～2019年 6月9日
3．毕业论文(设计)所需资料及原始数据（指导教师选定部分）

（1）了解基于B/S结构的Web编程有关知识和基本方法；
（2）了解数据库的基本知识及编程方法；
（3）了解高校教学过程及其管理信息的基本构成与特点；
（4）学习Android平台下Java语言及其网络编程方法。

4．毕业论文(设计)应完成的主要内容
（1）学习基于Android平台下编程有关知识和方法；
（2）了解高校教学过程及其管理信息系统基本数据结构与工作流程，用Java语言设计与实现基于Android平台下的教学过程信息管理APP；
（3）技术要求：
（i）完成该信息系统APP子模块、界面布局库的设计与实现；
（ii）编程实现移动端注册、登录至服务器端、用户管理等功能； 
（iii）实现移动端进行学生/教师手工录入/存储、修改/删除、查询等基本功能；
（iv）编程实现使系统具备师生互动等功能。 
5．毕业论文(设计)的目标及具体要求

论文应该包括方案的选择及论证
   论文应该包括实验数据结果与分析
   要求对毕业设计存在的问题作出说明，并提出改进的意见和改进方法
提交毕业论文
6．完成毕业论文(设计)所需的条件及上机时数要求


计算机、Android开发平台；
计算机上机≥120学时
任务书批准日期  2019 年 1 月 4日 教研室(系)主任(签字)           
任务书下达日期  2019 年 1 月 4日 指导教师(签字)                 
完成任务日期  2019 年 6 月 9日 学生(签字)                     

```
